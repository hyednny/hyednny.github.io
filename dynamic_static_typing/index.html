동적 타이핑과 정적 타이핑이란 ?

동적과 정적 타이핑 설명에 앞서 데이터 타입이란 값의 종류라고 표현한다.
데이터 타입은 값의 종류 (저장 가능한 메모리 크기)

  • 동적 타이핑(Dynamic typing)
    타입을 직접적으로 선언하지 않는다. (컴퓨터적 구조를 생략)
    코드를 작성하는 시간은 빠르고 로직을 파악하기 쉽지만 코드를 실행하는 속도가 느리고 큰 프로젝트에서 심각한 오류가 날 수도 있다.

  • 정적 타이핑(Static typing)
    데이터 타입을 직접 명시해준다.
    코드의 구조를 파악하기 쉽고 큰 프로젝트에서 적합

상황에 따라 둘 다 사용하지만 정적 타이핑을 통해 타입에 대한 권한을 가지는게 더 좋은 코딩 방식이다.
